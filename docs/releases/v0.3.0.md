# rss-agent-skills v0.3.0

Release date: 2026-05-22

Status: released

## Highlights

- Adds `subscription-research-agent` as the local-first research orchestration Skill.
- Adds the `packages/research-cli` Node/TypeScript CLI for subscription research workspaces.
- Introduces SQLite-backed research workspace persistence, including schema migrations.
- Archives RSS evidence into local workspace storage while keeping final reports Agent-written.
- Persists RSS ingest runs with criteria, worker stats, source-health summaries, archive counts, and entity-link counts.
- Adds conservative commentary-source and original-source attribution for archived articles and evidence briefs.
- Persists per-source health observations across ingest runs and adds `subscription-research source-health`.
- Carries forward the `v0.2.0` RSS Skills suite scope: `rss-source-curator`, digest presets, keyword groups, topic assignment, and grouped Markdown output.

## Release Notes

This release moves the project from an RSS Skills suite toward a local-first subscription research Agent toolkit. The Node/TypeScript CLI is now the shared deterministic RSS implementation, while the research workspace adds SQLite-backed evidence ingestion, entity extraction, source-health history, and evidence brief generation.

The CLI still does not generate final research reports by itself. Agents should synthesize final daily reports from evidence briefs using the `subscription-research-agent` daily report contract.
